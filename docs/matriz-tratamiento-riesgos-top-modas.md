# Matriz de tratamiento de riesgos - Top Modas

Sistema evaluado: aplicacion web de gestion para Top Modas, con modulos de autenticacion, roles, caja/ventas, inventario, solicitudes de stock, devoluciones, promociones, reportes, boletas PDF y base de datos PostgreSQL con Prisma.
